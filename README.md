# Netflix Dashboard Analyzer

## Overview
Analyzed Netflix dataset to extract insights on content trends, genres, ratings, and growth.

## Tech Stack
- Python
- Pandas
- Matplotlib / Seaborn

## Key Insights
- Most content added after 2015
- Drama and Comedy dominate
- Movies > TV Shows

## Features
- Data cleaning & preprocessing
- Visual dashboards
- Trend analysis

## How to Run
pip install -r requirements.txt
python main.py
